# Music Explorer 🎵

Frontend project focus only that actually talks to a real API. You can search for any song or artist and it'll pull live results from iTunes album art, song info, and a 30 second preview you can play right in the browser.

## What it does

- Search any artist or song
- Browse results as cards with album art
- Play a 30-second preview of any track
- Now playing bar at the bottom while audio is running

## Built with

- HTML, CSS, vanilla JavaScript
- [iTunes Search API](https://developer.apple.com/library/archive/documentation/AudioVideo/Conceptual/iTuneSearchAPI/index.html) (free, no key needed)

## How to run it

Download the files, open `index.html` in your browser. That's it no installs, no setup.

## Notes

This is a learning project. I built it to understand how frontend apps communicate with real APIs. Some songs won't have a preview available that's a limitation of the iTunes API, not a bug.
